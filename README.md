# FFMind
A command tool for generating MindMap immediately
## Examples
```
*Content //FFMind Content
RootNode
-SubNode1
--SubSubNode1
--SubSubnode2
--SubSubNode3
--SubSubNode4
-SubNode2
--SubSubNode1
--SubSubNode2
--SubSubNode3
-SubNode3
--SubSubNode4
--SubSubNode5
--SubSubNode6
<Render Above
*Global
TL,TC,TR //TL - text left, TC - text center, TR - text right
%16,Arial //Font Size,Font Family
#FF0000,#FF0000 // First is Foreground Color, Second is Background Color
*Main
-Foreground Color,Background Color|FontSize,FontFamily|Align Left,Align Middle,Align Right



```
